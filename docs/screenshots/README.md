# Screenshots

This directory contains screenshots of the Monarch Learning Platform.

## Adding Screenshots

1. Take screenshots of key features:
   - Dashboard
   - Content Management
   - Assessment Generation
   - Tutor Bot
   - Analytics

2. Save screenshots as PNG files:
   - `dashboard.png`
   - `content.png`
   - `assessment.png`
   - `tutor-bot.png`
   - `analytics.png`

3. Optimize images for web (recommended):
   ```bash
   # Using ImageMagick
   convert screenshot.png -resize 1920x1080 -quality 85 screenshot.png
   
   # Or using online tools like TinyPNG
   ```

4. Update the README.md to reference your screenshots.
